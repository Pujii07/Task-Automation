# 🤖 Task Automation with Python Scripts

## 📌 Project Overview

This project was developed as part of the **CodeAlpha Python Programming Internship**. The aim of this task is to automate a small real-life repetitive activity using Python scripting.

In this project, the automation script automatically detects and moves all **.jpg image files** from a source folder to a destination folder. This reduces manual work and demonstrates how Python can automate file management tasks efficiently.

The project uses Python libraries such as `os` and `shutil` to perform file operations and automate the workflow.

---

## 🎯 Objective

The objective of this project is to:

- Automate repetitive file handling tasks
- Move all `.jpg` files from one folder to another
- Create destination folders automatically if they do not exist
- Reduce manual effort through Python automation

---

## ✨ Features

✔ Automatically detects `.jpg` files  
✔ Moves files from source folder to destination folder  
✔ Creates destination folder if not available  
✔ Displays moved file names  
✔ Shows total number of files moved  
✔ Simple console-based interface  
✔ Beginner-friendly implementation
